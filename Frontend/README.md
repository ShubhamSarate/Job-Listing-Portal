# Job Portal Frontend

This is a minimal React + Vite frontend scaffold with Bootstrap CSS and two pages:

- `Login` (src/pages/Login.jsx)
- `Signup` (src/pages/Signup.jsx)

Run locally:

```powershell
cd "Frontend"
npm install
npm run dev
```

Open the shown local address (usually http://localhost:5173).

Notes:
- Pages use Bootstrap via CDN (index.html). Replace with local `bootstrap` package if preferred.
- Forms are placeholders; connect to your backend endpoints when ready.
